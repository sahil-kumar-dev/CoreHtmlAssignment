Ordered list of html tags

  1. head- It contains all the meta data and title of the page.

  2. a- It defines a link.

  3. br- It adds a line break.

  4. article- It defines a article of the page.

  5. main- It defines the main content of the web-page.

  6. body- It contains the body of the web-page.

  7. img- It used to add a image to the web-page

  8. i- It is a formatting tag and is used to itaclic a line.

  9. div- It used to wrap html elements.

  10. hr- It is used to add a horizontal line to the web-page.

  11. strong- It defines important text.

  12. table- It is used to add a table to the web-page.

  13. th- It is used to add table heading.

  14. td- It is used to add table data.

  15. video- It is used to add video to the web-page.
